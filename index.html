FancyVoice

1.0版本，只能进行基本的对话操作，暂无命令式口令操作。

Download this project as a .zip file Download this project as a tar.gz file

sdk采用科大讯飞的语音云接口
开发语言Java
开发平台Android Studio
关于如何集成sdk可以参考科大讯飞在极客学院上的官方教程
参考郭霖《第一行代码》使用ListView控件制作聊天界面
聊天气泡使用Adobe Photoshop CS6制作
语义理解产生的xml结果使用Pull方式进行解析
侧边栏效果的实现是将SlidingMenu的库作为module集成到VoiceHelper中
由于使用了语音合成功能;需要安装科大讯飞应用语记才能进行语音交互
